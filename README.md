## Deployment Documentation

### 1. Google AI Studio to GitHub Connection
This application was generated using Google AI Studio App Build. The source code was synced directly to this GitHub repository utilizing the native **"Sync to GitHub"** integration within the AI Studio interface. No manual source-code downloads or Google Cloud publish features were used, adhering strictly to the assignment guardrails.

### 2. Cloudflare Pages Deployment
This repository is connected and deployed via Cloudflare Pages.
*   **Source:** GitHub Repository
*   **Production Branch:** main
*   **Framework Preset:** Vite (or None)
*   **Build Command:** `npm run build`
*   **Build Output Directory:** `dist`
